#### Deactivating Search Models using SAP Fiori app F3036 Manage Search Models
1. Open the SAP Fiori App: Manage Search Models to see all available CDS View-based search models and their activation status.
2. Find the search model.

    <img width="1185" height="367" alt="image" src="https://github.com/user-attachments/assets/3fa932c8-c8e8-43ca-854d-38e8c72bcf70" />

> [!NOTE]  
> The new version is `CD$ALL~ESH_U_ACTYTYPEV2~` Deactivate earlier version: `CD$ALL~ESH_U_ACTIVITY_T~`.

3. Expand the search model you want to deactivate.
4. Switch the Enabled toggle button from Yes to No.
5. Confirm you want to disable the search.
6. Refresh your web browser and check the unwanted search is no longer shown in the search objects list.
